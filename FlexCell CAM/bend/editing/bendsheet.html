Bends spreadsheet

You can do a detailed editing of all settings for all bends by opening the Bend Settings spreadsheet. To access this spreadsheet select the settings icon from the command bar, and select Bend Settings (or use the shortcut key sequence T, A).

The number of bend column in the Bend Settings depends on the number of bends in the bend part.
Table 1. Basic

Field

Description

Inner angle

The inner angle of bending.

Length

Total bend length.

Press force

Computed press force.
Table 2. Ram Cycle
Field Description

Closing speed

The speed of the beam up to the mute point.

Mute point

At the mute point, the max speed ("Rapid down") is automatically switched over to the reduced speed ("Clamping"). The mute point is above the top edge of the lower tool. "Mute distance 1" programmed sheet thickness = distance between the tip of the upper tool and the upper edge of the lower tool.

Clamping speed

The speed of the beam between the mute point and the clamping point.

Clamping point correction

Adjust the value to compensate for variations in sheet thickness.

Pinch delay

The delay is mostly required by automated machines. During the waiting time, the beam will stop at the clamping point.

Bend speed

The speed of the beam during bending.

Press delay at BDC

The beam will remain at the lower dead point for this duration.

Decompress distance

With the decompression path, the tension in the machine’s frame is released in a controlled fashion after a bend, as it is under maximum tension at the lower dead point (LDP). The decompression path moves the beam upwards at a slow speed.

Decompress speed

Speed of the beam along the decompression path (decompression).

Opening height

The beam opening is the measure by which the beam moves upward after a bending operation.
Positive value: Press beam is positioned above the mute point.
Negative value: Press beam is positioned below the mute point.

Opening speed

Speed of the beam after the decompression path (decompression).

Field Description

Retract direction

Here you select in which direction the back gauge is to be withdrawn.

Retraction 1 and Retraction 2

Distances by which the gauges 1 and 2 move away from the gauging position once the part is clamped.

Z2 position

X1 position

X2 position

X position of gauge 2

R1 position

R2 position
