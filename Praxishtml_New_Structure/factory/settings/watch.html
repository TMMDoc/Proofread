Watch

Watch settings

Enable 'Live Folder' - Set this option to enable the live folder.

Watch this folder - Set this as the location to which Praxis will watch.

Watch sub-folders also - Enable this option for Praxis to also check subfolders within the location stated above.

After File Import

After import - After the file is imported the user has two options, the file can be deleted from the location or moved to a different location.

Move files to this folder - If the above option is to move the file to a different location, enter here the location to which the file should be moved.

Move spreadsheets with errors to this folder - If a spreadsheet is found to have errors, this can be moved to a different location which can be stated in this section.

Spreadsheet import settings

Import spreadsheet from 'Live Folder' - Enable this option to allow praxis to import spreadsheets found in the live folder location.

Do not import spreadsheet with errors - Enable this option for Praxis not to import spreadsheets with errors. They can be moved to a different location, as stated above, to be edited.

Spreadsheet import option - This option can set a default import type which can be any of the following options based on your available license.

  • Import Jobs

  • Import Orders

  • Import Assemblies
