Prerequisites

Praxis setup displays the below mentioned warning message at the beginning of Installation if the Praxis setup is run without first running the Prerequisites. Pre-Requisite installation is necessary only for the first time in Praxis Server/Client and can be ignored thereafter.

  • Execute the PreSetup.Praxis.<Version>.exe to install all the Pre-Requisites required to run Praxis application.

Version is Year with alphabet suffix.

  • Click Install to begin the Installation.

  • This setup installs all native redistributable, .Net Frameworks and other drivers used by Praxis.

  • Click Finish to complete the setup and Restart the computer.
