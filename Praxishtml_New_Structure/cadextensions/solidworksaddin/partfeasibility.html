Check feasibility

The part feasibility feature in the Praxis SolidWorks Add-in allows users to verify whether a sheet metal part can be successfully produced using the available factory machines and tooling configurations in Praxis.

This functionality replicates the same analysis steps that the Praxis engine performs during actual part upload but runs locally—without modifying or replacing any existing parts in the Praxis part library.

  • The feasibility dialog displays the analysis results for all available factory machines configured in Praxis.

  • Each machine’s status is shown as:

    • Feasible - The part can be processed successfully.

    • Warning - The part can be processed with limitations (for example, bending or cutting constraints).

    • Error - The part failed feasibility due to a blocking issue.

  • Click on icon to open the evaluated technology file (Flux or MetaCAM) for detailed inspection.

For assemblies, the same functionality performs a comprehensive feasibility analysis for each individual sheet metal component.

  • The results are displayed in the Assembly Feasibility dialog, listing each part’s material, configuration, and feasibility status.

  • Selecting a part displays its detailed information on the right panel, including material, size, weight, and feasibility result.

  • The feasibility analysis for assemblies is performed part by part, using the same logic as individual part feasibility checks.

  • The process stops automatically at the first blocking error where further progress is not possible.

  • Only sheet metal parts with valid materials and parameters are analyzed.

  • Non-sheet metal components are skipped but still listed for reference.

  • Since the feasibility run does not save or upload data to Praxis, it can be safely used for early-stage design validation and “what-if” analysis scenarios.

You must have a valid Flux or MetaCAM license to open and review the generated technology files linked in the feasibility results.
