Program Creation and Loading Part

Program Creation and Loading Part

1. Homing the Axes:

  • Open RA on the machine controller (HMI).

  • Check the Machine ID, it should be 84 while bending. To change the machine, modify it in the path “C:\ProgramData\Metamation\RA\RData\Middleware”

  • Navigate to Settings → Jog Axes and start the homing process.

  • If the emergency stop is pressed, an error will appear on the HMI. Reset the E-Stop button and Press the Up pedal on the foot pedal to reset the error.

  • After clearing the error and when the machine is ready for homing a pop-up message will appear on the HMI: "Press Start to complete homing process."

  • Press the green illuminated Start button on the controller panel to initiate homing. Homing will proceed for each axis one by one.

  • The '*' symbol in front of all axis values indicates that the respective axes are not yet homed.

  • Once homing is complete, the '*' symbols will disappear for all axes.

2. Robot Mode Activation:

Switch ON the Robot Mode in the bending machine after successful homing.

3. Loading and Running the Program:

  • Open the Run page in RA.

  • Click Transfer then Import and open the required .fx file of the robot bend operation in RA.

  • Verify the punch and die tooling by next tool and prev tool; modify them as per user requirements if necessary, in edit page.

Before running the program, ensure that in the robot teach pendant no previous bending programs are active. If any are open, close them and open a non-relevant robot bending program to ensure proper robot program loading.

4. Tooling and Bend Sequence Modifications:

In RA, go to Edit Page:

  • In Sequence, Change the bending sequence if required. It is advisable to change when using multiple bends if it causes any errors.

  • In Gauges, Adjust the flat (1 or 2) gauge settings accordingly.

  • In Tooling, select or modify or delete the punch and die tools as per bending requirements.

  • After making changes view the Simulation, to verify bending operations.

  • Click Done after all adjustments.

5. Running the Program:

  • Go to the Run page and click Run. Then Program page opens.

  • Auto mode should selected when running the program.

  • In Edit Cycle:

    • Modify the Pinch Adjustment (Pinch Adj.) value if necessary to adjust the pinch point.

    • Set Step Change to TDC+External if the program includes more than one bend.

    • Copy these settings to all bends.

  • In Correction:

    After bending measure the bend angle and verify if the actual and expected bend values are same if any changes required directly give the angle in measured interior angle then it will automatically calculate the correction which needs to be given.

6. Tooling Verification and Setup:

Check Machine Tooling:

Ensure the punch and die is clamped properly and match the tooling defined in the program. If changes are required:

  • Open the side view bar in RA.

  • Select Clamp/Unclamp Punch.

  • Use the Up pedal to perform the clamping or unclamping process.

  • Ensure tools are properly clamped and secured in the rail before starting the bending.

7. Manual Run for Testing program in Robot:

  1. Verify Tool Number: Ensure the correct tool number is set in the program.

  2. Robot Mode Settings:

    • Set the robot to T1 Mode and activate Step Mode in the Teach Pendant (TP).

    • Maintain a low robot speed during manual operation for safety.

    • Check for any errors in TP, if exist reset all the errors. Then start the program.

  3. Deadman Switch Requirement: The Deadman Switch must be pressed (enabled) while operating manually.

  4. Clearing System Messages:

    • Clear any pop-up messages such as "Robot Mode Active" or "Robot Mode Inactive" displayed on the HMI.

    • These messages appear when the Deadman Switch is enabled/disabled.

    • Reset them using the Foot Pedal before proceeding.

  5. Error Check Before Program Start: Ensure all system messages and errors are cleared before starting the robot program.

  6. Program Execution:

    • On the machine HMI, confirm the pop-up message: "Program not started in the robot."

    • Manually start the robot bending program from the Teach Pendant (TP).

    • Open and run the Main Bending Program in Step Mode which will have the same name as the .fx file.

  7. Machine Status Check: Verify that there are no errors displayed on the machine HMI before running the robot program.

  8. Position Verification:

    • Check the initial approach point, contact point, and ensure the sheet is being properly picked by the gripper.

    • Verify the mute point and apply back gauge corrections in the HMI if necessary.

  9. Ram Movement Control: For ram movement, press the Up Pedal as the machine will pause at the clamping point, waiting for the user confirmation to proceed.

  10. Bend Completion and Return:

    • Turn OFF Step Mode during the bending operation to allow the robot to move smoothly and simultaneously for accurate results.

    • Once the bend is completed, turn ON Step Mode again.

    • From there, the robot will transition to the Deposit Sub-Program, place the part down, and return to the Home Position, with each step executed manually via the Teach Pendant.
