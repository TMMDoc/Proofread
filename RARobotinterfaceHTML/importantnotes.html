Creating a Bending Program for Robot Mode

Creating a Bending Program for Robot Mode

Following are the important notes for creating a bending program for robot mode:

  • The control must not be blocked in robot mode or it will not be possible to carry out any bending.

  • Withdrawal should be active in each bending step to preserve the two added switches. A small withdrawal path of approx. 5mm is sufficient.

  • The downward movement in press operation from the clamping point to the lower dead point, which in turn bends the sheet, must always wait for the withdrawal of the backgauge.

  • The step change from one bending step to the next is always performed at the upper dead point.

  • For this reason, bending programs can only be run in automatic mode by robots in AUTO mode! MANUAL mode is not designed for this purpose.

  • If multiple foot switches are present in the configuration, only the first foot switch may be selected at any time. The robot can only switch foot switch 1 via the robot interface.

  • The optoelectronic safety device must always be deactivated.

  • Foot switch 1 must always be plugged in at the terminal strip of robot interface -X13 and never at the original position parallel to the robot interface on the panel terminal strip -X5. This always applies, even if the key switch was used to disable robot mode.
