Reference

Precondition

Press beam is located under the reference point.

When location of the press beam is under the reference point, the axes can move to the reference point via IO bits according to the following sequences.

If the press beam is located above the reference point, the referencing method mentioned above will lead to the following Error message: “Switch Off Robot mode & continue homing using Foot pedal Down!”

In this case operator needs to firstly use the operating mode key switch to switch to manual mode. Then operator needs to enter the bending cell and refencing from the press brake’s operator panel. After the axes move to the reference points, switch back to Robot mode.
