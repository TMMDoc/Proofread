Form detection

Form detection

Flux automatically identifies punch forms located on a single layer within a sheet metal model. When a surface model is converted to a metal model, all formed areas are recognized and converted into punch forms.

Import a sheet metal (having formed areas) into Flux. Select Convert to Sheet-metal from workflow panel.

If the punch form has already been identified, it is highlighted in light green, otherwise it is yellow in color.

Modify forming

In order to detect a formed area in a part:

  1. Click on any yellow highlighted formed area to access the Forming panel.

    Figure 1. Forming panel

  2. Select Identify…​ option to access Add Forming panel.

    1. Make Hole: The punch form is converted into a hole.

    2. Delete: The punch form is deleted.

    3. Similar: This function selects all forms which are similar in appearance to the selected form.

  3. Adjust the settings as required in the Forming panel.

    Figure 2. Add Forming

    1. Show: All other forms which are already defined are also displayed here. This form can be displayed as reference by clicking on a name. To go back to adding the new form, you must click on *NEW* in the top of the list.

    2. Preview: Shows a preview of the selected form.

    3. Variants: The same name as the name for an existing form can be used. This is normal as there different ways to model a form. If several forms with the same name are added, they become variants. Forms with several variants have their quantity listed after their name as Instances (ex: 3). If one of these forms is selected, different variants can be displayed. All of these different variants have the same name and the same ID. They are only depicted by different forms. If the software detects one of these forms in an assembly, this form will be marked with this name and this ID.

      It is possible that two different forms with the same name have been grouped incorrectly. In such a situation, the Separate button can be used to separate the specific variants with a new name. Conversely, if two forms which originally had different names are to be grouped, simply select one of them and rename it so that it coordinates with the other one. Both are grouped together as variants of the same form.

    4. Name: This name is used to display the form.

    5. Identifier: An ID can be entered here.

    6. Output Layer (DXF): This setting is used to specify a specific layer at which this form is output. This is used by the punch CAM system to assign a suitable punching tool.

  4. Press Add button to save the form to database for later use.
