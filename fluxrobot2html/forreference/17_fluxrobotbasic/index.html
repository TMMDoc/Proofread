FluxRobot Basic

FluxRobot Basic

A simple and robust method to start with is to get a skeleton program for the robot and the machine. - The offline programming does not need to be as accurate, you reteach the significant points on the robot

The robot outputs basically a simple motion sequence

/PROG  Robo
/ATTR
OWNER		= NCCONVER;
COMMENT		= "Wolfs test prog";
PROG_SIZE	= 817;
CREATE		= DATE 24-05-15  TIME 11:02:10;
MODIFIED	= DATE 24-05-15  TIME 11:02:10;
FILE_NAME	= \O0001.N;
VERSION		= 0;
LINE_COUNT	= 8;
MEMORY_SIZE	= 1157;
PROTECT		= READ_WRITE;
TCD:  STACK_SIZE	= 0,
    TASK_PRIORITY	= 50,
    TIME_SLICE	= 0,
    BUSY_LAMP_OFF	= 0,
    ABORT_REQUEST	= 0,
    PAUSE_REQUEST	= 0;
DEFAULT_GROUP	= 1,*,*,*,*;
CONTROL_CODE	= 00000000 00000000;
LOCAL_REGISTERS	= 0,0,0;
/APPL
/MN
1:  !,Program to pick a part ;
2:  UFRAME_NUM=1 ;
3:L P[1:PickApproach] 1000cm/min FINE;
IO[]=1..
4:L P[2:PickContact] 1000cm/min FINE;
5:L P[3:PickSeparate] 1000cm/min FINE;
6:L P[4:PickLiftClear] 1000cm/min FINE;
7:L P[5:PickStartSwing] 1000cm/min FINE;
8:L P[6:PreBendSafe] 1000cm/min FINE;
9:L P[7:FrontOfStation] 1000cm/min FINE;
10:L P[8:PartInsert] 1000cm/min FINE;
11:L P[9:DownMute] 1000cm/min FINE;
12:L P[10:DownPinch] 1000cm/min FINE;
13:L P[11:GaugeRetract] 1000cm/min FINE;
14:L P[12:Bending] 1000cm/min FINE;
15:L P[13:OverBending] 1000cm/min FINE;
16:L P[14:Decompress] 1000cm/min FINE;
17:L P[15:UpOpen] 1000cm/min FINE;
18:L P[16:PostBendSafe] 1000cm/min FINE;
19:L P[17:PostBendSafe2] 1000cm/min FINE;
20:L P[18:PartInsert] 1000cm/min FINE;
21:L P[19:DownMute] 1000cm/min FINE;
22:L P[20:DownPinch] 1000cm/min FINE;
23:L P[21:GaugeRetract] 1000cm/min FINE;
24:L P[22:Bending] 1000cm/min FINE;
25:L P[23:OverBending] 1000cm/min FINE;
26:L P[24:Decompress] 1000cm/min FINE;
27:L P[25:UpOpen] 1000cm/min FINE;
28:L P[26:PostBendSafe] 1000cm/min FINE;
29:L P[27:DropAloftFore] 1000cm/min FINE;
30:L P[28:DropAloftAft] 1000cm/min FINE;
31:L P[29:DropSafe] 1000cm/min FINE;
32:L P[30:DropApproach] 1000cm/min FINE;
33:L P[31:DropRelease] 1000cm/min FINE;
34:L P[32:DropRetreat] 1000cm/min FINE;
35:L P[33:DropHome] 1000cm/min FINE;
36: END ;
/POS
P[1:"PickApproach"]{
GP1:
    UF : 1, UT : 1,
J1 =     168.81  deg,	J2 =    46.87  deg,	J3 =     -89.62  deg,
J4 =     0.000 deg,	J5 =     0.000 deg,	J6 =     0.000 deg
};
