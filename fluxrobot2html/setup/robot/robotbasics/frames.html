Frames

Frames is a coordinate system that helps the robot to understand where things are placed in its workspace. They tell the robot how to move in the correct direction during working in the different setups.

Frames tell the robot where the origin is, and which way is X, Y, and Z. Using frames allows you to program the robot easily and accurately in different scenarios.

Types of Frames

  1. Tool Frame (UTOOL)

  2. User Frame (UFRAME)

  3. Jog Frame

  4. Cell Frame

Tool Frame

  • The Tool Frame defines the position and orientation of the Tool Center Point (TCP).

  • The TCP is a specific point in space that represents the tip of the tool. Since the robot only knows its wrist position, user need to calculate the TCP to tell the robot where the tool tip is located.

  • Setting the Tool Frame ensures the robot moves and rotates the tool accurately for tasks like picking, welding, or cutting.

User Frame

External to the robot - stays fixed on the floor, machine, or part.

Teaching the User Frame

When setting a User Frame, user can teach:

  • The origin point (zero position) of the table or part.

  • The X and Y directions to define the frame orientation. Even if the X and Y angles are not perfectly aligned, it will not affect the robot’s working as long as the frame is set properly for the process.

Jog Frame

Used for manual movement in custom directions

How to Setup or Verify Frames in FANUC Robot Using the Teach Pendant (TP)

To setup or verify frames in the robot, follow these steps on the Teach Pendant (TP):

Steps

  1. Press the MENU button on the Teach Pendant.

  2. Navigate to SETUP → Frames.

  3. Press F1 (Type) or select Others to view the list of available frame types.

  4. The following frame options will appear:

    • Tool Frame (UTOOL)

    • User Frame (UFRAME)

    • Jog Frame

    • Cell Frame

  5. Select the frame you need to setup or verify.

  6. Click on Method and select the calibration method that is most suitable for your setup.

    • Choose between 3-point, 4-point (XY/XZ), or 6-point for TCP calibration.

    • For User Frames, select 3-point or 4-point method based on frame like machine or Tilt pallet or Centering Table.
